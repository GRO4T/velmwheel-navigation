robot_localization
==================

robot_localization is a package of nonlinear state estimation nodes. The package was developed by Charles River Analytics, Inc.

Please see documentation here: http://wiki.ros.org/robot_localization

robot_localization_api
======================

This directpory is a fork of the ***galactic*** branch of the original [robot_localization](https://github.com/cra-ros-pkg/robot_localization/tree/galactic) 
package. The for has been provided to include API header files o the package into the install set of the package to enable custom implementations of 
the filtering nodes. All changes to the package has been marked with `@modified` or `@added` tag comments.
